## ⚠️ 사용 목적 안내

이 퀴즈 샘플은 **퀴즈 형식 및 구조 참고용**으로만 제공됩니다.

- 샘플 퀴즈의 주제는 **자바 기초** (JVM, 클래스/객체, 컬렉션, 예외처리 등)입니다.
- 실제 과제에서 생성해야 하는 퀴즈의 주제는 **제공된 STT 스크립트와 커리큘럼**을 기반으로 해야 합니다.
- 샘플을 그대로 사용하지 말고, **문제 유형 · 보기 구성 · 정답 형식**을 참고하여 새로운 퀴즈를 생성하세요.

## 퀴즈 세트 구성

총 3개 세트, 40문항으로 구성되어 있습니다.

| 세트명 | 문항 수 | 문제 유형 | 설명 |
|---|---|---|---|
| 기초평가 | 10문항 | 4지선다 | 자바 기초 개념 평가 |
| 자바기초 | 20문항 | 5지선다 | 자바 기초 복습 퀴즈 |
| 자바기초 재평가 | 10문항 | 5지선다 | 자바 기초 재평가 |

## 문서 구조 안내

각 문항은 아래 3가지 영역으로 구성됩니다.

```
┌──────────────────────────────────────┐
│ Q1  문제 내용                         │  ← 문항 번호 + 문제
├──────────────────────────────────────┤
│  1  보기 1                            │
│  2  보기 2                            │  ← 보기 (4지 또는 5지)
│  3  보기 3                            │
│  4  보기 4                            │
├──────────────────────────────────────┤
│ 정답  N번 — 정답 텍스트               │  ← 정답 (번호 + 내용)
└──────────────────────────────────────┘
```

## 실제 퀴즈 생성 시 참고사항

퀴즈 생성 시 아래 기준을 따르세요.

- **주제**: 제공된 STT 스크립트 및 `curriculum_kdt-backendj-21th.csv`의 `content`, `learning_goal` 필드 기반
- **문제 유형**: 샘플과 동일하게 객관식 형식을 기본으로 하되, 추가 유형 구현 가능
- **보기 수**: 4지선다 또는 5지선다 선택 가능
- **배점**: 문항당 10점 기준 권장
- **문항 수**: 강의 1일치 기준 최소 5문항 이상 생성 권장
