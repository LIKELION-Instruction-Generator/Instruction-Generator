# 1. 프로젝트 개요

본 문서는 멋쟁이사자처럼 심화과정(NLP/CV) 수료생을 대상으로 운영하는 로켓단 인턴십 프로젝트의 과제 명세서입니다. 각 과정별 2개 과제를 제시하며, 팀별로 1개 과제를 선택하여 4주간 수행합니다.

프로젝트는 단순 기술 구현이 아닌, 실제 클라이언트(AXP)의 문제를 해결하는 실무형 프로젝트로 설계되었습니다. 평가 또한 코드 리뷰가 아닌 클라이언트 관점에서 서비스의 완성도, 커뮤니케이션 능력, 문제 해결 접근법을 중심으로 진행됩니다.

# 2. NLP 과정: AI 엔지니어 자연어처리과정

## NLP 과제 2: AI 복습 퀴즈 & 학습 가이드 생성기

*강의 내용 기반 맞춤형 퀴즈와 핵심 요약 학습 가이드를 자동 생성하는 시스템*

| 구분 | 내용 |
| --- | --- |
| 과정 | AI 엔지니어 자연어처리과정 (NLP) |
| 팀 구성 | 4인 1팀 (7~8팀) |
| 기간 | 4주 (총 60시간) |
| 운영 방식 | 100% 원격 |
| 핵심 키워드 | 퀴즈 자동 생성, 핵심 개념 추출, 학습 보조 도구 |

### 1. 프로젝트 배경 및 목표

교육 플랫폼에서 수강생의 학습 효과를 극대화하려면 강의 후 복습이 핵심입니다. 그러나 매 강의마다 양질의 퀴즈와 학습 가이드를 수작업으로 제작하는 것은 비용과 시간이 많이 소요됩니다. 본 프로젝트는 강의 스크립트에서 핵심 개념과 학습 포인트를 자동으로 추출하고, 주관식/객관식/빈칸 채우기/코드 실행 등 다양한 형태의 퀴즈와 해설을 생성하는 시스템을 구축합니다. 나아가 주차별 학습 가이드와 핵심 요약을 자동으로 제작하여 수강생에게 제공하는 것을 목표로 합니다.

### 2. 제공 데이터

| 데이터 유형 | 상세 내용 | 비고 |
| --- | --- | --- |
| 강의 스크립트 | STT 기반 강의 텍스트 (N주치분) | 텍스트 파일 |
| 강의 커리큘럼 | 주차별 학습 목표 및 커리큘럼 | 문서 |
| 기존 퀴즈 샘플 | 과목별 기출 퀴즈 (참고용) | 문서 |


### 3. 권장 기술 스택

| 구분 | 기술/도구 | 용도 |
| --- | --- | --- |
| LLM | OpenAI GPT-4o / Claude API | 퀴즈 및 학습 가이드 생성 |
| NLP 프레임워크 | LangChain | RAG 파이프라인, 프롬프트 체이닝 |
| 벡터DB | ChromaDB / FAISS | 강의 내용 임베딩 및 검색 |
| 웹 프레임워크 | Streamlit / FastAPI | 퀴즈 서비스 인터페이스 |
| 데이터 처리 | Python (pandas, KoNLPy) | 텍스트 전처리 |

### 5. 중간/최종 평가 산출물

| 구분 | 제출물 | 평가 포인트 (클라이언트 관점) |

### 7. 확장 가능성 (보너스)

수강생 오답 패턴 분석을 통한 개인 맞춤형 추가 학습 추천, 퀴즈 난이도 적응형 조절 알고리즘, LMS 연동을 통한 자동 퀴즈 배포 시스템 등으로 확장 가능합니다.
