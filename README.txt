Week 1 private runtime data bundle.

How to use:
1. Clone the code repository.
2. Copy the contents of this bundle into the repository root, preserving paths.
3. Run:
   ./ .venv_quizsvc/bin/python scripts/check_runtime_data_bundle.py --week-id 1
   ./scripts/run_api.sh
   cd frontend && npm run dev

Included:
- curriculum csv
- week 1 prepared corpus json files
- accepted week 1 topic/guide/quiz/report artifacts
